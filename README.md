# MovieMint

MovieMint is a full-stack web application built for learning and practicing **MERN stack development**.  
It combines **React + Tailwind CSS** for the frontend with **MongoDB, Express, Node.js** backend functionality via Next.js API routes.

## Features

- **Movie Data Fetching:** Uses OMDb API key to fetch movie details dynamically.  
- **Authentication:** Login and registration implemented with **JWT** for authentication and **Bcrypt** for secure password hashing.  
- **Frontend & Backend:** Frontend built with **React + Tailwind CSS**; backend implemented using **MERN stack** through Next.js API routes.  
- **Responsive UI:** Clean and modern interface with **Tailwind CSS**.  

## Tech Stack

- **Frontend:** React + Tailwind CSS  
- **Backend:** Node.js + Express (via Next.js API routes)  
- **Database:** MongoDB  
- **Authentication:** JWT + Bcrypt  
- **API Integration:** OMDb API  

## Purpose

The main goal of this project is to **strengthen full-stack MERN skills**, including:

- API integration  
- Authentication & security  
- Database connection with MongoDB  
- Frontend development with React and Tailwind CSS  

This project is intended for **learning and portfolio purposes**.
